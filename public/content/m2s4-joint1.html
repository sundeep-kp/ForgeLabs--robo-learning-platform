Build Joint 1 (1-DOF Prototype)

A robotic arm is a series of mechanical joints. We will prototype the first joint, often called the **base or waist joint**, which provides a single degree of freedom (1-DOF) crucial for horizontal rotation.

Diagram: Base Joint Assembly

This joint uses a heavy-duty servo (like an MG996R) connected to a 3D-printed or laser-cut base and turntable.

#include <Servo.h>
Servo baseJoint;

void setup() {
  baseJoint.attach(10); 
  baseJoint.write(90); // Start centered
  delay(500);
}

void loop() {
  // Rotate slowly from 0 to 180
  for (int angle = 0; angle <= 180; angle += 1) {
    baseJoint.write(angle);
    delay(10); 
  }
  delay(1000);
  
  // Rotate back slowly
  for (int angle = 180; angle >= 0; angle -= 1) {
    baseJoint.write(angle);
    delay(10);
  }
  delay(1000);
}
