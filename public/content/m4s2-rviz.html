Module 4, Session 2: RViz Visualization

What You'll Learn

This session introduces RViz, a tool for visualizing robot models, sensor data, and simulation results in ROS2.

Step-by-Step Instructions

  1. Install RViz using your package manager or ROS2 tools.
  2. Launch RViz and load a sample robot model.
  3. Explore basic visualization features: camera view, sensor data, robot state.
