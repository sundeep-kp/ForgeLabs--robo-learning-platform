Module 3, Session 4: Build a 2-DOF Arm

What You'll Learn

This session guides you through building a simple 2-degree-of-freedom robotic arm using servos and basic materials.

Step-by-Step Instructions

  1. Gather materials: two servos, base, arm segments, screws.
  2. Assemble the base and attach the first servo for rotation.
  3. Mount the second servo for the elbow joint.
  4. Wire both servos to Arduino and test movement.
