Multi-Servo Control (2‚Äì4 DOF)

To create the remaining joints (shoulder, elbow, wrist), you need to control multiple servos simultaneously. This requires managing multiple PWM outputs and coordinating their timing in your Arduino code.

Diagram: Multi-Servo Wiring

Always ensure all servos share a **common ground (GND)** with the Arduino, even if they draw power from an external source.

üß† Section 1 ‚Äî Core Concept: Servo Timing

The Arduino Servo library handles the timing of multiple PWM pulses seamlessly in the background. You simply declare a separate servo object for each joint:

  • `Servo baseJoint;`
  • `Servo shoulderJoint;`
  • `Servo elbowJoint;`

Using `delay()` in your main loop will halt all movement and calculation. In real automation, we must use non-blocking timing (like the `millis()` function) for complex coordination.

üîß Section 2 ‚Äî Hands-On: Adding More Joints

For your full robotic arm prototype, you will likely need to control at least 4 joints for a basic reach and grab function.

  1. Wire the shoulder servo signal to **Pin 11**.
  2. Wire the elbow servo signal to **Pin 12**.
  3. Ensure the **external power supply** can handle the current draw of all servos moving at once (stall current can be high!).

üíª Section 3 ‚Äî Arduino Code Example (Coordinated Move)

This code moves all three joints simultaneously to a target position.


#include <Servo.h>
Servo base, shoulder, elbow;

void setup() {
  base.attach(10);
  shoulder.attach(11);
  elbow.attach(12);
  // Start position
  base.write(90);
  shoulder.write(90);
  elbow.write(90);
  delay(1000);
}

void loop() {
  // Move to the "open" position
  base.write(150);
  shoulder.write(30);
  elbow.write(120);
  delay(2000);

  // Move to the "closed" position
  base.write(30);
  shoulder.write(150);
  elbow.write(60);
  delay(2000);
}
